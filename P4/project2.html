Structured Cabling and Network Setup

Project Objectives

  • Perform Cable Termination of UTP Cat6 Cables
  • Perform Cable Punch Downs on Patch Panel
  • Mount Switches and Patch Panel
  • Demonstrate Structured Cabling Between Devices
  • Test Network Connectivity (Ping)

1. Cable Termination of UTP Cat6 Cables

Cut cables to length, strip the jacket, untwist and arrange wires using the 568B standard. Crimp RJ45 connectors on each end with a crimping tool.

2. Cable Punch Downs on Patch Panel

Secure each cable's end into the patch panel using the 568B standard and a punch-down tool. Mount the panel to the rack and test connections using a cable tester.

3. Mounting Switches and Patch Panel

Secure the Cisco switch and Linksys router onto the 4U Networking Rack below the patch panel using screws.

4. Establishing Structured Cabling Between Devices

Connect terminated cables between the patch panel, keystone jack surface mount boxes, the switch, and the wireless router for a complete structured cabling setup.
