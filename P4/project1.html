Cloud Infrastructure Project (Microsoft Azure)

Overview

Designed and deployed a secure, cloud-based application environment simulating a business infrastructure using Microsoft Azure.

No screenshots for Security purpose

1. Virtual Network and Subnet Configuration

Configured a virtual network with segmented subnets (Web & DB), implemented Network Security Groups (NSGs) with customized inbound rules, and enabled NSG Flow Logs for monitoring.

2. Virtual Machines and IAM Setup

Provisioned and secured Windows Server VMs across availability zones, implemented managed identities, and assigned custom IAM roles for role-based access control.

3. Web App Deployment with Private Endpoints

Deployed an Azure Web App in a Python environment with Private Endpoints, integrated it with a Virtual Network and Azure Private DNS Zone for internal-only access.

4. Validation and Security Testing

Validated RDP access, VM interconnectivity, DNS resolution, and external access controls to ensure compliance and security.

5. CIS Benchmark Hardening

Applied an additional CIS Benchmark hardening policy to enhance cloud security posture.
